# Remedy

## Getting Started

### Install
```
git clone https://github.com/jm-shi/Remedy.git
cd Remedy
npm install
```

### Start
```
npm start
```

### Sass
To compile Sass to CSS and minify the output, run the following:
```
gulp sass
```

To watch for changes in Sass files, run the following:
```
gulp sass-watch
```
